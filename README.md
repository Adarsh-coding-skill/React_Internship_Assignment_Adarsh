# Frontend Technical Assessment

##  Project Overview
This project is a **Frontend Technical Assessment** completed using **React.js, HTML5, CSS3, and JavaScript**.  
The goal of the assessment was to demonstrate skills in:
- Building responsive and interactive UIs using React.
- Integrating APIs and dynamically displaying data.
- Writing clean, reusable, and maintainable code.
- Following best practices in frontend development.

---

##  Tech Stack
- **Frontend:** React.js (Functional Components + Hooks)
- **Styling:** CSS3, Flexbox, Grid
- **State Management:** React `useState` & `useEffect`
- **API Integration:** REST API calls using `fetch`/`axios`
- **Build & Deployment:** Netlify 
- **Version Control:** Git & GitHub
